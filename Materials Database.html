The application

Search it, edit it, shape it to fit you

From a powerful material search to a tabbed editor and a drag-and-drop display designer, the app put engineers in control of their own data views. Click any screen to enlarge.

Capabilities

More than a table of parts

A real engineering tool — request, search, customize, template and govern.

🔎

Material search

Rich, multi-field search across class, identifier, program and security level with sortable, exportable results.

📝

New-material requests

A request queue lets engineers submit new materials for review and entry into the shared library.

🎛️

Customizable display

Drag-and-drop field groups let each user shape the data view to their discipline and workflow.

🧾

Output templates

Define and apply output templates — including scientific-notation formatting — for clean, consistent reports.

🔗

Document & supplier links

Attach document and supplier URLs to any material, keeping spec sheets and sourcing a click away.

🔐

Access & security

Per-material access levels and security keep sensitive engineering data with the right people.
