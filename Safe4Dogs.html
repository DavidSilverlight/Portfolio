A classroom disguised as a dog app

Safe4Dogs is open source on purpose. I built it as a hands-on mentoring tool for new .NET MAUI developers — a friendly, real-world codebase where every feature doubles as a clean, readable lesson in how the framework's core pieces fit together.

LESSON 01

MVVM

Separate UI from logic with bindable, testable view models.

LESSON 02

Shell & Navigation

Structure the app and move between pages the MAUI way.

LESSON 03

Handlers

Customize native controls through the handler architecture.

LESSON 04

Dependency Injection

Wire up services cleanly and keep code testable.

LESSON 05

Value Converters

Shape data for the UI without polluting your models.

LESSON 06

Behaviors

Attach reusable UI logic without code-behind.

LESSON 07

Localization

Speak every dog-parent's language.

LESSON 08

Read the source

Open, commented and approachable — learn by exploring real code.
